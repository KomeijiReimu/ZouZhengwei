# 《汽车租赁系统V4.0》

> 压缩包内文件：
>
> - 项目源代码
> - 实验报告
> - 系统说明书
> - 汇报ppt
> - sql脚本
> - 打好的jar包
> - 功能结构图，类图等在说明书内



项目jdk：1.8

使用idea编写

使用Maven构建，使用Mybatis框架，使用Mysql数据库。

默认数据库配置信息为：

```xml
                <property name="driver" value="com.mysql.cj.jdbc.Driver"/>
                <property name="url" value="jdbc:mysql:///rentcar_sy?useSSL=false"/>
                <property name="username" value="root"/>
                <property name="password" value="1234"/>
```

Mysql版本8.0以上
